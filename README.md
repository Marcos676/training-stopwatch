# Training Stopwatch
Es un proyecto destinado asistir a los usuarios en sus entrenamientos, proporcionando un cronómetro para medir el tiempo de cada ejercicio.
## Características
- Cronómetro para medir el tiempo de cada ejercicio.
- Interfaz sencilla y fácil de usar.

## Tecnologías Utilizadas
### Cliente
- Vite
- React
- TypeScript
- Tailwind CSS

### Servidor
